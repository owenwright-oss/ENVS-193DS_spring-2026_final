# ENVS-193DS_spring-2026_final

Final for ENVS 193DS at UCSB

## General information

This repository contains the Final for the course ENVS 193DS at UCSB. The assignment includes research writing and interpretation of a published ecological study, a logistic regression analysis examining that of bird nest occurrence as a function of tree height and acacia ant species presence, and a simple linear regression analysis examining the relationship between sleep duration and productivity level using my personal data collected throughout Spring 2026.

## Data and file information

The code folder contains the Quarto file used to write and render the Final in Rstudio.

The data folder contains the data files used in the assignment, including the bird nest occurrence dataset, provided for me, and my personal sleep and productivity dataset, downloaded from google sheets.

The images folder contains image files used in the assignment, containing an image of my final affective visualization.

## Rendered output

The rendered final output is included as a PDF generated from the Quarto document. The final PDF contains the written responses, annotated R code, code output, statistical analyses, model diagnostics, figures, captions, and affective visualization materials completed for the ENVS 193DS Final.
